# Redmi/Xiaomi Setup Guide - Wire Auto Messenger

## Exact Steps for Redmi Note 10 Pro / Redmi Devices

### The Exact Option You Need to Enable

**Look for:** `Wire Auto Messenger` under **"Downloaded apps"** or **"Installed services"**

**NOT these:**
- ❌ TalkBack
- ❌ Select to Speak
- ❌ Switch Access
- ❌ Voice Access
- ❌ Any other system accessibility options

**YES, this one:**
- ✅ **Wire Auto Messenger** (under Downloaded apps/Installed services)

---

## Step-by-Step for Redmi Devices

### Step 1: Open Accessibility Settings

1. In Wire Auto Messenger app, tap **"Enable Accessibility Service"**
2. Or go manually: **Settings** → **Additional Settings** → **Accessibility**

### Step 2: Find the Right Section

On Redmi devices, you'll see multiple sections:

- **Vision** (TalkBack, etc.)
- **Hearing** (Sound Amplifier, etc.)
- **Physical & motor** (Switch Access, etc.)
- **Downloaded apps** ← **LOOK HERE!**
- **Installed services** ← **OR HERE!**

### Step 3: Find Wire Auto Messenger

1. Scroll down to **"Downloaded apps"** or **"Installed services"**
2. Look for **"Wire Auto Messenger"** in that list
3. It should show the app icon and name

### Step 4: Handle Restricted Settings (IMPORTANT!)

**If you see "Restricted Settings" blocking access:**

1. Go to **Settings** → **Apps** → **Manage apps**
2. Find **"Restricted Settings"** app
3. Tap on it
4. Look for **"Wire Auto Messenger"** in the restricted list
5. **Allow** or **Enable** access for Wire Auto Messenger
6. Return to Accessibility Settings
7. Try enabling again

**OR:**

1. Go to **Settings** → **Additional Settings** → **Privacy**
2. Find **"Restricted Settings"** or **"App Restrictions"**
3. Allow **"Wire Auto Messenger"**

### Step 5: Enable It

1. Tap on **"Wire Auto Messenger"**
2. You'll see a toggle switch
3. Toggle it **ON**
4. A warning dialog will appear
5. Read it and tap **"Allow"** or **"OK"**

### Step 6: Verify

1. Return to Wire Auto Messenger app
2. The status should show: **"✓ Enabled"** (green badge)
3. If it still shows "Not Enabled", try:
   - Force close the app
   - Re-open it
   - Check again
   - Make sure Restricted Settings allows the app

## Troubleshooting for Redmi

### Problem: Can't Find "Wire Auto Messenger"

**Solution:**
1. Make sure the app is installed
2. Go to: **Settings** → **Apps** → **Manage apps**
3. Find **"Wire Auto Messenger"**
4. Make sure it's installed and enabled
5. Then go back to Accessibility settings

### Problem: Toggle is ON but App Says Not Enabled

**This is usually Restricted Settings blocking access!**

**Solution:**
1. Go to **Settings** → **Apps** → **Restricted Settings**
2. Find **"Wire Auto Messenger"** and **Allow** it
3. Go back to Accessibility Settings
4. **Disable** the toggle
5. **Wait 2 seconds**
6. **Enable** it again
7. Tap **"Allow"** on the warning
8. Return to app and check

### Problem: Many Options, Don't Know Which One

**Look for:**
- App name: **"Wire Auto Messenger"**
- Under section: **"Downloaded apps"** or **"Installed services"**
- NOT under: Vision, Hearing, Physical & motor sections

### Problem: MIUI Security Blocking

**Solution:**
1. Go to **Settings** → **Apps** → **Manage apps**
2. Find **"Security"** or **"MIUI Security"**
3. Tap **"Disable"** (temporarily)
4. Enable Accessibility Service
5. Re-enable Security app

### Problem: Still Not Working

**Try This:**
1. Uninstall Wire Auto Messenger
2. Restart your phone
3. Reinstall the app
4. Follow setup steps again
5. Make sure to enable from **"Downloaded apps"** section

## Still Having Issues?

If you've tried everything and it's still not working:

1. **Check App Installation:**
   - Settings → Apps → Wire Auto Messenger
   - Make sure it's installed and not disabled

2. **Check MIUI Settings:**
   - Settings → Additional Settings → Privacy
   - Make sure app permissions are allowed

3. **Restart Device:**
   - Sometimes a restart helps refresh permissions

4. **Contact Support:**
   - Share a screenshot of your Accessibility Settings screen
   - We can help identify the exact option
